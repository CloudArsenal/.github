### What it does

- **Multi-account / multi-subscription scanning** — scan your entire org in one session
- **Live log streaming** — see every account as it runs, in real time
- **Plugin scanner architecture** — add a new scanner with a single Python file
- **Browser-native** — no CLI, no PowerShell, just `docker-compose up`
- **Export** — Excel, CSV, HTML reports out of the box
